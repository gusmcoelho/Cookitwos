# Guia Rápido do Projeto (Para a Apresentação)

Aqui está a sua "cola" super simples para lembrar o que cada arquivo faz na hora de apresentar:

- **`index.html`**: A página base (tela em branco) do site.
- **`main.jsx`**: O botão de ligar (puxa o código e injeta na tela em branco).
- **`App.jsx`**: O site de verdade (junta o menu, os produtos e gerencia a animação).
- **`node_modules/`**: Pasta com bibliotecas externas do sistema.
- **`public/`**: Arquivos de imagem, vídeo e a animação do cookie.
- **`src/`**: O coração do projeto, nosso código fonte.
- **`src/components/`**: Os pedaços da tela (como os botões, o rodapé, etc).
- **`package.json`**: Lista de compras (quais tecnologias o site usa).
- **`vite.config.js`**: O empacotador (faz o site rodar rápido).
- **`.github/`**: O robô que publica o site na internet sozinho.
